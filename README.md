# Controle Financeiro em Python

Sistema de controle financeiro desenvolvido em Python no terminal.

## Funcionalidades

* Adicionar receitas
* Adicionar despesas
* Visualizar movimentações
* Filtrar receitas e despesas
* Ver saldo atual
* Tratamento de erros com try/except

## Tecnologias utilizadas

* Python
* JSON
* Git/GitHub

## Objetivo

Projeto criado para praticar lógica de programação, modularização e manipulação de dados em Python.
